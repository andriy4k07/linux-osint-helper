Maigret (https://github.com/soxoj/maigret) - форк усіма відомої утиліти Sherlock від @soxoj для пошуку імен користувачів у соціальних мережах і на різних сайтах.
Usbsas - это инструмент для безопасного чтения информации с ненадежных USB-накопителей.
EagleEye - OSINT модуль, який дає змогу знаходити людей у соціальних мережах за фото і передбачуваним ім'ям.
ProxyBroker - це інструмент з відкритим вихідним кодом, який асинхронно знаходить публічні проксі з декількох джерел і паралельно перевіряє їх.
Zmap-ProxyScanne - швидкий і безпечний спосіб пошуку проксі-серверів. Знайдіть 2000-5000 робочих http,socks4,socks5 проксі за одне сканування.
Build-your-own-botnet - Фреймворк з відкритим вихідним кодом для створення власних ботнетів
SqliSniper - Це надійний Python-інструмент, призначений для виявлення сліпих SQL-ін'єкцій у заголовках HTTP-запитів, що базуються на часі. 
Mimesis - інструмент для генерації фейкових даних у великих кількостях.
Updo - це інструмент для моніторингу продуктивності веб-сайтів. Він надає метрики в реальному часі про стан сайту, час відгуку та багато іншого.
Prying Deep - інструмент для збору інформації в darknet.
Orca - фреймворк для целевого OSINT.
Snoop - один із найперспективніших OSINT-інструментів із пошуку нікнеймів
Deep Live Cam - це інструмент для заміни облич у реальному часі та створення відеодипфейків з використанням лише одного зображення.
KickThemOut - інструмент, що дає змогу відключати пристрої від мережі WiFi і користуватися всією пропускною здатністю для себе.
Telepathy-Community - OSINT-інструментарій для дослідження чатів у Telegram.
